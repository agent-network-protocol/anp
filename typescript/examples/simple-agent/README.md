# Simple Agent Example

This example demonstrates the basics of creating an ANP agent.

## What This Example Shows

- Creating a DID:WBA identity
- Creating an agent description
- Adding information resources and interfaces
- Signing the agent description
- Signing and verifying data

## Running the Example

```bash
npm install
npm start
```

## Expected Output

The example will:
1. Create a new DID identity
2. Create an agent description with metadata
3. Add information and interface definitions
4. Sign the agent description
5. Demonstrate data signing and verification

## Next Steps

After running this example, you can:
- Publish the DID document to make it resolvable
- Host the agent description at a public URL
- Implement HTTP endpoints for the agent interface
- Explore the authentication example for secure communication
